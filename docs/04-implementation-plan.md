# 04. Plan Trien Khai

## Phase 0: Chot thiet ke

Muc tieu:

- Chot rule chia tien.
- Chot DB schema ban dau.
- Chot API can lam truoc.

Ket qua:

- Tai lieu trong `docs/` on.
- Chua can code.

## Phase 1: Backend toi thieu

Stack:

- FastAPI.
- SQLAlchemy.
- PostgreSQL.
- Pydantic.
- uv.
- Docker Compose.
- Alembic.

Viec can lam:

1. Tao project backend.
2. Tao ket noi database.
3. Tao models.
4. Tao API members.
5. Tao API nop quy.
6. Tao API lich co dinh.
7. Tao API preview/chot buoi choi.
8. Test bang Swagger.

## Phase 2: Frontend toi thieu

Stack:

- Next.js.
- TypeScript.
- CSS module hoac global CSS don gian.

Man hinh can co:

1. Dashboard tong quan.
2. Thanh vien.
3. Nop quy.
4. Lich co dinh.
5. Tao buoi choi.
6. So quy.

## Phase 3: Auth va phan quyen

Them:

- User login.
- Password hash.
- JWT/session.
- Middleware bao ve API.
- UI theo role.

## Phase 4: Hoan thien nghiep vu

Them:

- Sua/huy buoi da chot bang giao dich dieu chinh.
- Loc giao dich.
- Export Excel/CSV.
- Bao cao theo thang.
- Nhac nguoi sap het quy.

## Phase 5: Deploy

Them:

- PostgreSQL.
- Alembic migration.
- Docker.
- Deploy backend.
- Deploy frontend.
- Backup database.
